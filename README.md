# CSH
A simple shell program completly written in C.

## Instalation
To download and install just run the following commands:
```bash
git clone https://github.com/mrehfeld-code/CSH;
cd CSH;
cmake -S . -B build;
cmake --build build;
sudo cmake install;
```

If everything gone well you can start the shell with:
```bash
CSH
```
